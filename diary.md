# 記録日記

* gitで管理したいディレクトリに入ってgit initを実行すると.gitって名前の隠しディレクトリが生成され、そのディレクトリがgitの管理対象となる。
* gitでいまいるブランチ名を知りたいときのコマンドはgit branch --show-current
* commitの対象とする(ステージに載せる)ファイルやディレクトリを選ぶコマンドはgit add ~~
* git statusを実行するとbranch名とステージングエリアに載ってるファイルがわかる
* git commit -m でメッセージ付きの記録(コミット)ができる
* git clone url でリモートリポジトリ(url)のディレクトリをローカルに保存できる
* open-duck はGitHub上のリポジトリ自体の名前で、origin と hogehoge は、あなたのローカルPC上でそのリモートリポジトリを指し示すために付けている「ニックネーム」や「ラベル」のようなもの
